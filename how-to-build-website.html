How to Build & Host This Website

October 26, 2025 • By Rabi Ibrahim

Have you ever wanted to create a sleek, professional portfolio or landing page but didn't know where to start? In this tutorial, I'll walk you through exactly how I built the RabiBytes website using vanilla HTML, CSS, and JavaScript, and how you can host it for free using GitHub Pages.

Step 1: Setting Up the Structure (HTML)

HTML (HyperText Markup Language) is the backbone of any website. For this project, we need a clean structure consisting of a Header (navigation), a Main area (Hero, Gallery, About), and a Footer.

Here is the skeleton of our index.html:

<!DOCTYPE html>
<html lang="en">
<head>
    <title>RabiBytes</title>
    <link rel="stylesheet" href="index.css">
</head>
<body>
    <header>...</header>
    <main>
        <section id="hero">...</section>
        <section id="videos">...</section>
    </main>
    <footer>...</footer>
    <script src="index.js"></script>
</body>
</html>

Step 2: Styling with Modern CSS

To achieve the "Dark Tech" aesthetic, we use CSS Variables for color management and Flexbox/Grid for layout. Crucially, we use a mobile-first approach.

Define your variables at the root:

:root {
    --bg-dark: #121212;
    --primary: #0ea5e9;
    --accent: #84cc16;
    --text-main: #ffffff;
}

body {
    background-color: var(--bg-dark);
    color: var(--text-main);
}

Step 3: Adding Interactivity (JavaScript)

Instead of hardcoding every video card in HTML, we use JavaScript to generate them dynamically. This makes adding new content as easy as adding an item to an array. 

const videos = [
    { title: "My Video", url: "..." },
    { title: "Another Video", url: "..." }
];

const gallery = document.getElementById('video-gallery');
// Map through data and inject HTML
gallery.innerHTML = videos.map(v => `...`).join('');

Step 4: Hosting on GitHub Pages

This is the best partâ€”hosting is completely free!

  1. Create a GitHub account.
  2. Create a new repository named username.github.io.
  3. Upload your index.html, index.css, and index.js files.
  4. Go to Settings > Pages and ensure the source is set to the main branch.

That's it! Your site will be live at https://username.github.io in minutes.

Conclusion

Building a static site doesn't require complex frameworks. With just three files, we've created a responsive, performant, and good-looking landing page. Happy coding!
